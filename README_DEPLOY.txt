DEPLOY (Render with Docker)

1) Create a new GitHub repo.
2) Upload: app.py, requirements.txt, Dockerfile.
3) Render → New → Web Service → connect repo → Deploy.
4) Optional login (Settings → Environment):
   FLASK_SECRET= MelehMotorsSecret123!
   BASIC_AUTH_USER= meleh
   BASIC_AUTH_PASS= yourStrongPassword
5) On iPhone: open URL → tap “📷 Scan with Phone Camera”. It auto-fills the payload box.
6) Fill VIN/price/date → Generate PDFs → Download.
